## 📸 Overview

A fully interactive personal portfolio website built from scratch — no frameworks, no dependencies. Pure HTML, CSS, and vanilla JavaScript.

---

## ✨ Features

- 🎨 Dark / Light mode toggle
- ⚡ Animated hero with live particle canvas
- 📊 Project progress tracker (X / 50)
- 🔍 Filterable project grid by language
- 🔐 Dedicated Cybersecurity section with glitch effect
- 🖱️ Custom cursor with hover interactions
- 📱 Fully responsive / mobile friendly
- 📬 Contact form with animated success state
- 🔢 CTF Challenges counter (200+)

## 🚀 Getting Started

No install needed. Just open the file:

```bash
git clone https://github.com/ThejasRajamoney/ThejasRajamoney.github.io.git
cd ThejasRajamoney.github.io
open index.html
```

Or visit the live site directly:
```
https://thejasrajamoney.github.io
```

## 🔄 Updating Projects

All content is in the `CONFIG` and `PROJECTS` arrays at the top of the `<script>` tag in `index.html`. Just edit and push — GitHub Pages auto-deploys.

```js
const PROJECTS = [
  { title: "Your Project", lang: "JavaScript", desc: "...", status: "complete", github: "#", demo: "#" },
  // add more here
];
```
